1) По всем уникальным исполнителям из оценки
   Находим первую значимую для планирования задачу
   Если не нашли значимую, то берем любую первую задачу
   Если не нашли не одной задачи, то идем на следующего уникального исполнителя next 1)
   
   С задачи берем: 
      Нагрузку часов в день 
      Длительность в рабочих днях
      Дата начала задачи
   
   текущий найденный ресурс = -1
   текущая найденная дата начала = текущая дата + 20 лет
   
   2)По всем ресурсам находящимся в группе ресурсов с наименованием совподающим с обрабатываемым уникальным исполнителем
      3)Начиная с даты начала задачи по всем дням с шагом в один день пока не найдем 
         4)Начиная с текущей обрабатываемой даты на всю длительность задачи в рабочих днях
            Проверяем загруженность ресурса 
               - загруженность ресурса должна быть менее 4 часов
               - загруженность ресурса + Нагрузку часов в день по задаче не должна превышать 8 часов
            Если условия не выполняются, то период не подходит ищем следующую за проверенной датой дату по циклу 3 next 3)
         end 4
         Если текущая найденная дата начала > текущей обрабатываемой даты то
            текущая найденная дата начала = текущей обрабатываемой даты
            текущий найденный ресурс = текущей обрабатываемый ресурс
         переходим к следующему ресурсу next 2)
      end 3
   end 2
   
   если текущий найденный ресурс = -1 то  
      меняем ресурс задачи на текущий найденный ресурс
      меняем дату начала задачи на текущая найденная дата начала
   иначе
      выводим сообщение что не удалось найти уникального исполнителя
